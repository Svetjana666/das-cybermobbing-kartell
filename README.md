# Cybermobbing-Kartell Dokumentation

Dieses Repository dokumentiert eine Sammlung von Underground-Künstlern und Musikprojekten, die im Zusammenhang mit dem Cybermobbing-Kartell untersucht werden.

## 🎯 Projektzweck

Die Dokumentation dient als Archiv für Künstler und Projekte, die Verbindungen zum Cybermobbing-Kartell aufweisen. Dies schließt sowohl musikalische Werke als auch Online-Aktivitäten mit ein.

## � Struktur

Jede Datei enthält:
- **Basisinformationen**: Spotify-Daten, Hörerzahlen, Genre
- **Biografie**: Verfügbare Informationen zum Künstler
- **Online-Präsenz**: Social-Media-Profile und Webseiten
- **Kontext**: Relevante Verbindungen und Aktivitäten

## 🔍 Untersuchungsfokus

Die Dokumentation konzentriert sich auf:
- Underground-Künstler mit geringer Reichweite
- Emo Rap und Alternative Genres
- Online-Community-Verbindungen
- Cybermobbing-Aktivitäten

## � Daten

- **Künstler**: 80+ dokumentierte Profile
- **Genre**: Primär Emo Rap/Alternative
- **Reichweite**: Meist unter 50 monatliche Hörer
- **Status**: Aktive Untersuchung
